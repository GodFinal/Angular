AngularJS 简介

什么是 AngularJS?

AngularJS 是一个为动态WEB应用设计的结构框架。它能让你使用HTML作为模板语言，通过扩展HTML的语法，让你能更清楚、简洁地构建你的应用组件。它的创新点在于，利用 数据绑定 和 依赖注入，它使你不用再写大量的代码了。这些全都是通过浏览器端的Javascript实现，这也使得它能够完美地和任何服务器端技术结合。

AngularJS是为了克服HTML在构建应用上的不足而设计的。HTML是一门很好的为静态文本展示设计的声明式语言，但要构建WEB应用的话它就显得乏力了。所以我做了一些工作（你也可以觉得是小花招）来让浏览器做我想要的事。formatDate

数据绑定

AngularJS 扩展了 HTML

相对与原来的HTML的编写模式，AngularJS 对于DOM的操作可谓是超前的，它不再仅仅是通过JS/Jquery等操作DOM，而是有它自己的一套数据绑定方式。

首先，当然是要引用 AngularJS，其次使用ng-app来指定哪一块DOM区域是 AngularJS 的操作区域，这个节点可以是html、body或者某个其它标签。只有在ng-app所在标签下的所有 AngularJS 操作才能起作用。使用ng-model绑定标签的值到 AngularJS。这样在 AngularJS 脚本中通过ng-model的值就可以访问到这个标签的值，也可以通过它来修改标签的值。{{}}是 AngularJS 中的数据绑定表达式，可以直接绑定变量到标签中。

例子：


性别：
婚否：
您的信息： {{userName+" "+userSex+" "+isMarried}}

源码：

  1. <div class="code" ng-app="">  
  2.     <input type="text" placeholder="姓名：" ng-model="userName" />  
  3.     <br />  
  4.     性别：  
  5.     <label>  
  6.         <input type="radio" name="sex" value="男" ng-model="userSex" />男</label>  
  7.     <label>  
  8.         <input type="radio" name="sex" value="女" ng-model="userSex" />女</label>  
  9.     <br />  
  10.     婚否：  
  11.     <label>  
  12.         <input type="checkbox" ng-model="isMarried" />已婚</label>  
  13.     <br />  
  14.     您的信息： <span>{{userName+" "+userSex+" "+isMarried}}</span>  
  15. </div>  

简单控制器

AngularJS 控制器

ng-controller是 AngularJS 的应用程序控制器，通过脚本获取控制器后就可以随心所欲的操作 AngularJS 了。

$scope是控制器中 AngularJS 的模型对象，它就相当于一个作用域对象，通过它我们就可以去操作变量或者方法等了。

例子：


全名： {{fullName()}}

源码：

  1. <div class="code" ng-app="userNameApp" ng-controller="userNameCtrl">  
  2.     <input type="text" placeholder="姓：" ng-model="firstName" />  
  3.     <input type="text" placeholder="名：" ng-model="lastName" />  
  4.     <br />  
  5.     全名： <span>{{fullName()}}</span>  
  6. </div>  
  7. <script type="text/javascript">  
  8.     var userNameApp = angular.module("userNameApp", []);  
  9.     userNameApp.controller("personCtrl", function ($scope) {  
  10.         $scope.firstName = "God";  
  11.         $scope.lastName = "Final";  
  12.         $scope.fullName = function () {  
  13.             return this.firstName + " " + this.lastName;  
  14.         }  
  15.     });  
  16. </script>

例子：

  • {{$index+"."+user.name}}

  1. <ul ng-app="app" ng-controller="userListCtrl">

过滤器

AngularJS 过滤器

过滤器通过字符|添加到表达式中，不仅仅可以进行数据过滤和格式化，还可以进行数据转换。

过滤器 描述
currency 格式化数字为货币格式。
filter 从数组项中选择一个子集。
lowercase 格式化字符串为小写。
orderBy 根据某个表达式排列数组。
uppercase 格式化字符串为大写。

例子：

大写：{{goods[0].name|uppercase}}
小写：{{goods[0].name|lowercase}}
货币格式：{{goods[0].price|currency}}
过滤：{{goods|filter:'n'}}
排序：{{goods|orderBy:'price'}}

源码：

  1. <div ng-init="goods=[{name:'Apple',price:3.4},{name:'Orange',price:5.25},{name:'Bananer',price:2.5}]"></div>  
  2. <p>大写：{{goods[0].name|uppercase}}</p>  
  3. <p>小写：{{goods[0].name|lowercase}}</p>  
  4. <p>货币格式：{{goods[0].price|currency}}</p>  
  5. <p>过滤：{{goods|filter:'n'}}</p>  
  6. <p>排序：{{goods|orderBy:'price'}}</p>
